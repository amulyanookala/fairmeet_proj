# FairMeet - AI Powered Fairness and Engagement Analytics for Online Meetings

FairMeet is a web-based application that analyzes meeting transcripts and audio to evaluate participation, fairness, and engagement.

## Features
- Transcript and audio input
- Speaker-wise analysis
- Fairness, Agenda, Engagement scores
- Charts and visual insights
- Save meeting history
- PDF report generation

## Tech Stack
- Python
- Streamlit
- SQLite
- NLTK
- Matplotlib

## Run the Project
```bash
pip install -r requirements.txt
streamlit run app.py
